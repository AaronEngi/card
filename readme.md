参考weui示例工程的代码结构，定个规范吧。
代码目录结构
src目录是源代码
component下面是各种组件，页面上的小组件，可以复用，放在这个目录下面。
pages 是各个页面，实际上是按页面分的功能模块。相同模块的多个页面，也可以放在同一个文件夹下。
app.js是入口

build目录是编译后的代码
index.html 不是编译文件，是源代码
